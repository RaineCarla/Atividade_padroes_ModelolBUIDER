Projeto de exemplo utilizando o padrão de projeto Builder em Java.

O sistema permite montar pedidos personalizados de hambúrguer ou pizza,
com validações no método build() para evitar pedidos inválidos.

Foram implementados testes no Main com:
- Hambúrguer válido
- Pizza válida
- Pedido com pagamento em dinheiro e troco
- Pedido inválido demonstrando exceção
